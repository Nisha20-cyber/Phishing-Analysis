# Task 2: Phishing Email Analysis

## Objective
To analyze a suspicious email and identify phishing characteristics.

## Tools Used
- Sample phishing email (text file)
- Manual analysis

## Files Included
- phishing_email.txt (sample phishing email)
- phishing_analysis.txt (analysis report)

## Phishing Indicators Identified
- Spoofed sender email address
- Urgent and threatening language
- Suspicious link
- Generic greeting
- Request for sensitive information

## Outcome
Learned how to identify phishing emails and understand common phishing tactics.
